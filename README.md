Dataset Summary

The dataset used in this project is the "Students Performance in Exams" dataset. It contains information about students such as gender, parental level of education, lunch type, test preparation course, and their scores in math, reading, and writing.

Machine Learning Problem: Classification

The goal of this project is to classify whether a student will pass or fail in math based on their characteristics and performance indicators.

Target Variable: Pass/Fail (derived from math score)

A student is considered to pass if the math score is 50 or above, and fail if the score is below 50.
